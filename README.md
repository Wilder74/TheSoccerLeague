The project we worked on is a soccer league simulation. We where given multiple rules to follow and things to include in the program.
Including weather, number of goals scoored, teams, etc. Basically the program makes 4 teams and runs a simulation os a season.
The season depends on the weather. If the temperatures are 32 degreed F. or lower the teams won't play because is too cold. After three
weeks where the temps are too cold to play the season is over. In the other hand if the weather is better meaning the temps are hotter.
The teams have the possibility to perform better and score more goals.

We used different files for the code. The Main one where we run the simulation of the season. We also have the games build
there, we also include the temperatures and scores we generated and our print statements.
We also had the Team file wich stored information about each team including things like wins,looses,ties,etc.
And the Game file wich stored the game ID, temperature and both teams scores.

For us to be able to do all that we did things lie intantiate the team objects, store that in a ArrayList. Create one to store all games
played. We also looped the season to keep running until we had the three weeks of freezing temps to stop the code. We used a random generator,
for the games, temperature, etc. 
In general we used many different "concepts" such as: methods,ArrayList, randomizer, switch statements, loop, getters & setters, constructors.
